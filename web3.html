Page web dynamique avec JavaScript

Durée : 60 heures – Projet achevé

Contexte

En renfort chez ArchiWebos pour le site portfolio d’une architecte d’intérieur, j’ai développé trois fonctionnalités clés en JavaScript, en partant du HTML/CSS existant et en communiquant avec une API back-end. 

De : Charlotte  
À : Moi  
Objet : Brief ArchiWebos  

– Figma du design  
– Code back-end & front-end statique  
– Kanban des tâches

Ce que j’ai réalisé

  • Page de présentation dynamique des projets : filtres, galerie interactive
  • Page de connexion administrateur from scratch avec validation des champs
  • Modale d’upload de médias : sélection de fichiers, preview avant envoi
  • Appels AJAX à l’API pour récupérer et poster les données JSON
  • Gestion des événements utilisateurs et manipulation du DOM
  • Versioning Git et tests en environnement local

Compétences acquises

  • Programmation JavaScript (ES6): modules, fetch, async/await
  • Manipulation du DOM et gestion des événements
  • Communication sécurisée avec une API REST
  • Construction d’interfaces dynamiques modulaires
  • Intégration front-end Agile et collaboration UI/UX

Outils & technologies

  • JavaScript (ES6+), HTML5, CSS3
  • Fetch API pour requêtes HTTP
  • Visual Studio Code
  • Figma pour les maquettes
  • Git & GitHub pour versioning

Livrables

  • Code source (HTML/CSS/JS) structuré en modules
  • Notebooks de tests unitaires (Jest) et d’intégration
  • Documentation d’intégration API et guide d’utilisation
