Projet 13 – Détection de faux billets avec R / Python

Durée : 70 heures – Projet achevé

Contexte

L’ONCFM (Organisation nationale de lutte contre le faux-monnayage) m’a confié la conception d’un modèle de machine learning capable, après analyse de caractéristiques (longueur, largeur, épaisseur, etc.), de distinguer vrais et faux billets en euros.

Ce que j’ai réalisé

  • Exploration & visualisation des données de 1 500 billets (1 000 vrais, 500 faux)
  • Prétraitement : nettoyage, normalisation et sélection de features
  • Test de 4 algorithmes : K-means, régression logistique, KNN, Random Forest
  • Évaluation par cross-validation et comparaison des métriques (précision, rappel, F1)
  • Sélection et déploiement du meilleur modèle (Random Forest)
  • Développement d’une application de prédiction dans un notebook séparé
  • Rédaction d’un support final présentant la méthodologie, les résultats et recommandations

Compétences acquises

  • Construction de pipelines ML : EDA, feature engineering, modélisation
  • Comparaison et optimisation d’algorithmes supervisés et non supervisés
  • Validation et interprétation des métriques de classification
  • Packaging d’une application de prédiction (notebook interactif)
  • Communication des résultats à un public non technique

Méthodologie

  1. Exploration des variables et détection des anomalies
  2. Normalisation et codage des features numériques
  3. Implémentation de K-means pour segmentation exploratoire
  4. Entraînement & évaluation de régression logistique, KNN, Random Forest
  5. Choix du modèle final et analyse des importances de variables
  6. Construction d’un notebook “applicatif” pour prédiction en temps réel

Outils & technologies

  • Python (pandas, scikit-learn, matplotlib, seaborn) ou R (tidyverse, caret)
  • Jupyter Notebook / RStudio
  • Git & GitHub pour versioning des notebooks
  • Docker (optionnel) pour containerisation de l’application

Livrables

  • Notebook EDA & modélisation
  • Notebook applicatif de prédiction
  • Support de présentation (PDF/PPT)
  • Code source sur GitHub
