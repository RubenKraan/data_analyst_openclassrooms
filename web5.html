Débug et optimisation du site de photographe

Durée : 50 heures – Projet achevé

Contexte

Freelance pour Nina Carducci, photographe, sur son site nina-carducci.github.io. Mission : améliorer les performances, le SEO et l’accessibilité, corriger des bugs JavaScript.

Ce que j’ai réalisé

  • Audit initial avec Lighthouse (performance) et Wave (accessibilité)
  • Optimisation des images : compression, responsive et lazy loading
  • Minification et scindage du CSS/JS, injection de CSS critique
  • Ajout de balises SEO : meta description, Open Graph, JSON-LD Schema.org (LocalBusiness)
  • Référencement local : adresse, téléphone, horaires via Schema.org
  • Correction de la navigation modale (précédent/suivant) et du filtre de catégories
  • Amélioration de l’accessibilité : ARIA labels, rôle landmarks, contraste et navigation clavier
  • Rapports avant/après avec captures Lighthouse & Wave
  • Rédaction d’un cahier de recette et d’un rapport d’intervention détaillé

Compétences acquises

  • Performance web : audits, optimisation d’actifs et critical rendering
  • SEO on-page et données structurées (Schema.org)
  • Accessibilité WCAG 2.1 et tests Wave
  • Debug JavaScript avec Chrome DevTools
  • Rédaction de cahier de recette et documentation technique

Outils & technologies

  • Chrome DevTools, Lighthouse, Wave
  • ImageMagick / Squoosh pour compression d’images
  • Visual Studio Code
  • Git & GitHub pour versioning du code
  • JSON-LD pour balisage Schema.org

Livrables

  • Code optimisé et corrigé sur GitHub
  • Rapport d’audit avant/après (PDF) avec métriques et captures
  • Cahier de recette détaillant les tests fonctionnels
  • Guide d’optimisation SEO & accessibilité
