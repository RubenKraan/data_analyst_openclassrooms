Projet 15 – Application de location immobilière avec React

Durée : 60 heures – Projet achevé

Contexte

En freelance pour Kasa, leader de la location d’appartements entre particuliers, j’ai repris le front-end legacy ASP.NET pour le réécrire en React. Laura, la CTO, m’a fourni les maquettes Figma, les guidelines de code et un JSON simulant les 20 dernières annonces. Avec Vite et React Router, j’ai mis en place une SPA moderne, responsive et modulaire.

Ce que j’ai réalisé

  • Initialisation du projet avec Vite + React (ES6/Eslint/Prettier)
  • Configuration de React Router pour gérer trois pages principales (Accueil, Logement, 404)
  • Développement de composants : Header, Footer, Card, Gallery, Collapse
  • Implémentation de la galerie : navigation circulaire et découpe centrée des images
  • Création du composant Collapse avec état ouvert/fermé et animations CSS
  • Intégration des données JSON, gestion des routes dynamiques (useParams)
  • Tests de réactivité et optimisation pour mobile, tablette et desktop
  • Respect des coding guidelines Kasa et déploiement sur Netlify

Compétences acquises

  • Maîtrise de React (hooks, props, state) et structuration d’une SPA
  • Gestion des routes avec React Router v6
  • SASS pour styles modulaires et animations CSS
  • Appels et rendu de données statiques via JSON
  • Coordination front-end/UI design avec Figma et équipe produit

Outils & technologies

  • React & React Router
  • Vite (bundler ultra-rapide)
  • Node.js & npm
  • SASS pour la gestion des styles
  • Figma pour l’obtention des maquettes et ressources
  • Git & GitHub pour versioning et CI/CD

Livrables

  • Repo GitHub complet (setup, composants, styles, tests)
  • Application déployée en production (Netlify)
  • Documentation technique (README, guidelines, scripts)
  • Guide de déploiement et de maintenance
