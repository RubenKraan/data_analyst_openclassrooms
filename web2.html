Projet 1 – Intégration responsive du site Booki

Durée : 40 heures – Projet achevé

Contexte

Booki, une agence de voyage en ligne, a fourni des maquettes Figma pour mobile, tablette et desktop. Ma mission était d’intégrer leur page d’accueil à la main avec HTML & CSS, sans copier le code généré par Figma.

Ce que j’ai réalisé

  • Installation et configuration d’un environnement front-end (VS Code, live server)
  • Découpage précis des maquettes en sections HTML sémantiques
  • Développement d’un CSS responsive : media queries, flexbox et grid
  • Optimisation des images et gestion du chargement asynchrone
  • Versioning du projet avec Git et publication sur GitHub
  • Validation sur mobile, tablette et desktop pour garantir une UX fluide

Compétences démontrées

  • Intégration HTML5 & CSS3 conforme aux designs
  • Design responsive : flexbox, grid et media queries
  • Structuration et sémantique du code
  • Gestion de projet front-end et versioning Git
  • Collaboration avec un UI designer et la CTO

Outils & technologies

  • HTML5 & CSS3 (Flexbox, Grid)
  • VS Code + Live Server
  • Git & GitHub pour le suivi des versions
  • Figma pour interpréter les maquettes

Livrables

  • Code source complet (index.html, style.css, dossier images)
  • Pages validées sur mobile, tablette et desktop
  • Repository GitHub avec historique des commits
