Développement back-end pour Mon Vieux Grimoire

Durée : 60 heures – Projet achevé

Contexte

En partenariat avec Kévin (front-end React), j’ai conçu le back-end de « Mon Vieux Grimoire », un site de notation de livres pour une chaîne de librairies. Le cahier des charges précisait la création d’une API RESTful sécurisée, reliée à MongoDB.

Ce que j’ai réalisé

  • Configuration d’un serveur Express avec architecture MVC
  • Modélisation Mongoose des schémas : Users, Books, Ratings
  • Implémentation d’opérations CRUD sécurisées pour livres et notations
  • Système d’authentification JWT avec hash bcrypt pour les mots de passe
  • Gestion des uploads d’images et optimisation avec Sharp
  • Calcul automatique de la note moyenne des livres (aggregation pipeline)
  • Respect des bonnes pratiques Green Code : optimisation des requêtes et mémoire

Compétences acquises

  • Création d’API RESTful avec Node.js & Express
  • Modélisation et requêtes MongoDB via Mongoose
  • Authentification sécurisée (JWT, bcrypt)
  • Traitement et optimisation d’images côté serveur
  • Structuration MVC et organisation de projet

Outils & technologies

  • Node.js, Express
  • MongoDB, Mongoose
  • JSON Web Tokens (JWT) & bcrypt
  • Sharp pour optimisation d’images
  • Docker pour conteneurisation (optionnel)
  • Git & GitHub pour versioning

Livrables

  • Repo GitHub complet (code source, README, scripts de démarrage)
  • Documentation API (Swagger / Postman Collection)
  • Cahier de recette pour tester chaque endpoint
  • Guide de déploiement et optimisation Green Code
